# **App Name**: Linguistic Decoder

## Core Features:

- Word Analysis: Analyze the given word using the Seven-Voices solver to determine its primary and frontier paths.
- Voice Path Display: Display the primary voice path, ring path, level path, and checksums of the analyzed word.
- Frontier Paths Display: Display the frontier paths with their respective ring paths, level paths, and checksums, offering alternative phonetic interpretations.
- Language Candidate Mapping: Leverage Gemini 2.5 as a tool to map the solved voice path to potential language candidate families, providing insights into the word's linguistic origins.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to convey intellect, analysis, and historical connections.
- Background color: Light Grey (#EEEEEE), almost white, for clean readability and to give prominence to data displays.
- Accent color: Soft Amber (#FFB300) to highlight key analytical components and interactive elements.
- Headline font: 'Space Grotesk' sans-serif font for titles.
- Body font: 'Inter' sans-serif font to ensure clear presentation of analytical results.
- Code font: 'Source Code Pro' for API calls
- Use a clear, structured layout with distinct sections for word input, analysis results, and linguistic mapping. Prioritize readability and easy navigation.
