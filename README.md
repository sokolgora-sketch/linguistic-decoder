# Linguistic Decoder — Seven-Voices Engine

## Quick Start
- Node 20+
- Install: `npm i`
- Dev: `npm run dev`

## Engine
- Version: 2025-11-16-core-3
- Voices: A,E,I,O,U,Y,Ë
- Strict params: edge=0.25, ins=3, del=4

## Env
- .env.local:
  - GOOGLE_GENAI_API_KEY=...
  - GENKIT_API_KEY=...
  - NEXT_PUBLIC_ENGINE_VERSION=2025-11-16-core-3
